# Daily Report — 2026-06-23

## Summary

Fixed two issues causing the NDVI and Weather Rust micro-services to be unreachable from Django: a port mapping mismatch in the NDVI service and a DNS race condition in the Weather service.

## Changes

- `docker-compose.pilot.yml`: Fixed NDVI port mapping from `${NDVI_PORT}:8081` to `${NDVI_PORT}:${NDVI_PORT}` to match the service's actual listening port.
- `services/weather/src/db.rs`: Added retry loop (10 attempts, 2s backoff) for database connection to handle Docker DNS race condition on container startup.
- `services/weather/Cargo.toml`: Added `time` feature to tokio for `sleep()` in retry logic.
